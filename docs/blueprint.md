# **App Name**: Munir Gallery

## Core Features:

- Parallax Hero Section: Create a full-screen hero section with a WebP image sequence background that animates on scroll.
- Customizable Text Overlay: Allow the user to change the photo name, subtitle, and description displayed on the hero section.
- Theme Color Customization: Enable users to modify the brand accent color.
- Dark/Light Mode Toggle: Implement a toggle for switching between dark and light mode.
- Variant Navigation: Implement a PREV/NEXT navigation to switch between multiple variants of content (photo, text, color) and image sequences.
- WebP Sequence Management: Allow users to upload and manage multiple WebP image sequences for the parallax background.
- Loading Experience: Display a loading overlay with a progress bar before showing the hero and while switching between variants.

## Style Guidelines:

- Primary color: Black (#000000) for a premium, dark cinematic feel.
- Background color: Charcoal gray (#333333) for dark mode.
- Accent color: White (#FFFFFF) for contrast and highlighting interactive elements.
- Body and headline font: 'Inter', sans-serif, for a clean, modern look throughout the website.
- Special Title Font: 'Belleza', sans-serif, suitable for headline and shorter chunks of text
- Use minimal, monochrome social icons in the bottom center of the hero section and the footer.
- Full-screen hero section with overlay text block on the left and variant navigation on the right. Sticky top navigation bar and standard sections below the hero.
- Smooth parallax scrolling effect with WebP image sequence tied to scroll position. Fade-in/fade-out animation when switching between variants.
